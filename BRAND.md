# One Fourteen — Brand Spec

The visual operating system for everything One Fourteen prints, posts, builds, or hangs on a wall.

## Voice

Dark. Direct. Sharp. Late-night, slightly editorial.
No exclamation points. No hype words. No emoji. Headlines all-caps.

## Colors

| Name | Hex | RGB | Use |
|---|---|---|---|
| **Void Black** | `#0A0A0A` | 10, 10, 10 | Primary background. The bar's main palette. |
| **Cream** | `#E8E0D0` | 232, 224, 208 | Primary type on dark. Clean editorial. |
| **Aged Cream** | `#E0D5BC` | 224, 213, 188 | Yellowed cream for vintage/divey designs only. |
| **Brass** | `#B5953A` | 181, 149, 58 | Accent. Attribution lines, decorative dots, secondary text. |
| **Deep Red** | `#B91C1C` | 185, 28, 28 | Bold accent for music/nightclub direction. Use sparingly. |
| **Steel Gray** | `#8A9099` | 138, 144, 153 | Tertiary type only. Avoid on merch. |

## Typography

### Primary stack (Void System — editorial)

- **Bebas Neue Regular** — Display headlines, hero type
- **Barlow Condensed Medium** — Body, subtitles, addresses
- Use at scale, all-caps, tight tracking

### Alternate stacks (use intentionally, not by default)

| Stack | Use case |
|---|---|
| **Alfa Slab One** + **Special Elite** | Divey / vintage / worn dive-bar tees |
| **Allura** (script) | Music / nightclub / romantic late-night |
| **Anton** | Tour tees, dense rap-merch-style layouts |
| **Black Ops One** / **Permanent Marker** | Punk zine, ransom-note, DIY |

## Address lockup

The official address lockup that appears on most merch:

```
114 W MAIN ST  ·  LOUISVILLE  KY
```

Always use the middle-dot character (`·`, U+00B7) as separator. Never a hyphen.
Optional attribution line above:

```
ONE FOURTEEN  ·  WHISKEY ROW
```

## Design rules

- **Type only by default.** Add graphic elements only when they earn their place.
- **No gradients.** Solid spot colors only. DTG prints them cleaner and they stay on-brand.
- **Generous letter spacing on caps.** Two spaces between words inside lockups (e.g., `MUSIC  SOUNDS  BETTER`).
- **Designed for Void Black.** Every design assumes a dark background. If switching to a cream tank, invert.

## When to break the system

The Void System is the default. It's broken intentionally when the design needs a different emotional register:

- **Divey vibe** for vintage / DIY-feeling drops
- **Disco vibe** for music-night programming
- **Punk vibe** for one-off chaotic moments

Each break is documented in the design file. No silent deviations.

## Don't

- Don't use the cream and brass together as if they were a brand pair — brass is an accent only
- Don't put serif fonts and Bebas in the same lockup
- Don't lower-case headlines (except Allura script)
- Don't add stars / fleur-de-lis / Kentucky tourist iconography
- Don't print on heather/marled blanks unless intentionally going vintage
