# Whiskey Row Girl Activewear

Minimal women’s activewear capsule built around the phrase:

```text
WHISKEY ROW GIRL
```

## Product direction

This should feel more like premium activewear than bar souvenir merch:

- Small bone-colored mark
- Black base garments
- Minimal placement
- No oversized back graphics
- No cocktail clip art
