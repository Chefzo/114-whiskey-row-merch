## Refresh from source

If a font file is corrupted or you want the latest version:

```bash
curl -L -o BebasNeue-Regular.ttf \
  https://github.com/google/fonts/raw/main/ofl/bebasneue/BebasNeue-Regular.ttf
```

Replace the filename and path segment with the appropriate font from the table.
Google Fonts repo path format: `ofl/<lowercase-name>/<Filename>.ttf` (or `apache/<name>/...` for Apache-licensed).
